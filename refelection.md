## What challenges did you face when refactoring your code to use Bootstrap?
    I did not face much challenges as the code invloves just adding the right bootstrap classes in html file, that can be easily find in the bootstrap website.
## How did using Bootstrap utility classes and components simplify your styling process?
I  just added the classes in html file, instead of writing any css properties and attributes. This made the alignment easier than using custom css. Also, achieved the responsiveness with single utility class, which reduced my development time when I struggled with custom CSS.

## In what scenarios might you choose not to use Bootstrap and write custom CSS instead?
If the there is no bootstrap class for the color you would want use, it is good to use the custom css.
